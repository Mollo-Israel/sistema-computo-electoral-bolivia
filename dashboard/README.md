# Dashboard - Sistema de Cómputo Electoral

## Responsable: Erick Diaz

## Descripción
Dashboard web que compara los resultados del pipeline RRV (Recuento Rápido de Votos) con el Cómputo Oficial.

## Contenido planificado
- Participación ciudadana.
- Votos totales: válidos, blancos, nulos.
- Votos por partido.
- Estado de actas: recibidas, procesadas, pendientes, observadas, rechazadas, duplicadas, publicadas.
- Tasa de participación y porcentaje por partido.
- Margen de victoria.
- Velocidad de procesamiento.
- Diferencias entre recuento rápido y oficial.
- Análisis geográfico por departamento y municipio.
- Transparencia y acceso a actas digitales.
- Indicadores técnicos: latencia, throughput, disponibilidad, seguridad.
- Anomalías y logs.
