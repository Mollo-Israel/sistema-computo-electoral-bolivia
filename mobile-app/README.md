# App Móvil - Sistema de Cómputo Electoral

## Responsable: Ferrufino

## Descripción
Aplicación móvil para que los jurados de mesa puedan fotografiar y enviar las actas electorales al pipeline RRV.

## Funcionalidades planificadas
- Captura de foto del acta electoral.
- Envío de imagen al backend (`POST /api/rrv/acta/mobile`).
- Autenticación del jurado por PIN.
- Confirmación de envío exitoso.
